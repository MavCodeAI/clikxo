# Development Setup Guide

## Prerequisites
- Node.js 18.17.0+
- npm 9.0.0+
- Git

## Global Tools (Install Once)
```bash
npm install -g create-next-app typescript tailwindcss
```

## Project Dependencies (Per Project)
```bash
npm install
```

## Cache Configuration
```bash
npm config set cache "C:/Users/Darwesh/npm-cache"
npm config set prefer-offline true
```

## For Future Projects
1. Use `create-next-app` for new projects
2. Copy your favorite configuration files
3. Reuse components from this project
